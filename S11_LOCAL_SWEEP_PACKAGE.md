# S11 局部扫参方案说明

本包用于继续寻找更好的 S 参数折中解。

根据用户反馈：

- 模型结果的谐振频率更接近 `900 MHz`
- 历史真实样本的 S11 更深

所以这次不再全局乱搜，而是围绕这两组分别做小范围扰动。

## 扫参思路

### 1. 围绕历史最深 S11 样本

基础尺寸：

```text
2.25000, 20.0000, 2.25000, 75.0000, 75.0000, 1.20000, 80.0000, 80.0000
```

目标：

- 保留较深 S11
- 尝试把谐振频率往 `900 MHz` 附近拉

主要微调：

- `cut_x`
- `fw`
- `h1`
- `cut_y`

### 2. 围绕模型 900 MHz 解

基础尺寸：

```text
3.13591, 19.0690, 3.33152, 153.586, 150.702, 1.70934, 77.6789, 78.4607
```

目标：

- 保持频率接近 `900 MHz`
- 尝试进一步压低 S11

主要微调：

- `cut_y`
- `fw`
- `h1`
- `px`
- `py`

## 文件说明

- `s11_local_sweep_candidates.csv`：适合直接查看或复制到表格中
- `s11_local_sweep_candidates.json`：结构化版本

建议优先跑全部 20 组。若时间有限，可以先跑 `hist_*` 和 `model_base` 附近的前几组。
